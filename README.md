# 🏆 Software de Coleta e Análise de Dados de Desempenho em Partidas de Futsal

Este projeto é um sistema completo para **registrar, analisar e visualizar dados de desempenho em partidas de futsal**. Ele conta com uma aplicação frontend para interação do usuário e um backend que gerencia os dados e a lógica de negócio.

## 📦 Estrutura do Projeto

- `FRONTEND/` — Interface web construída com React.
- `BACKEND/` — API desenvolvida com Node.js e banco de dados.

## 🚀 Como Executar

1. Clone este repositório:

```
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Instale as dependências:

```
cd BACKEND
npm install

cd ../FRONTEND
npm install
```

3. Inicie os dois projetos:

```
npm run dev
```

> Certifique-se de que o backend e o frontend estão usando as portas corretas e se comunicando corretamente.

## 📊 Funcionalidades

- Registro de eventos como gols e finalizações
- Análise de desempenho por jogador e por equipe
- Visualização gráfica dos dados coletados
- Suporte a múltiplas partidas
